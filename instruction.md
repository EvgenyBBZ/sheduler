# Инструкция для нейросети (универсальная):
Тебе будет передан список сотрудников, распределенных по подгруппам (секторам), и шаблон расписания.
Твоя задача — расставить имена вместо маркеров `[Чел X]`, следуя простому правилу привязки.
Например, если для Г12 переданы 4 имени: "Иванов, Петров, Сидоров, Смирнов", ты привязываешь их по порядку:
- Чел 1 = Иванов
- Чел 2 = Петров
- Чел 3 = Сидоров
- Чел 4 = Смирнов

И просто заменяешь текст в шаблоне. **Не изменяй время и структуру, только подставляй имена.** В ответе выведи готовое расписание.
